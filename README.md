# Login & Sign Up UI Animado

Interfaz moderna desarrollada con HTML, CSS y JavaScript que permite alternar entre formularios de Login y Registro mediante animaciones deslizantes.

El proyecto utiliza un diseño minimalista y dinámico donde los paneles se desplazan horizontalmente para mostrar el formulario seleccionado, creando una experiencia visual más moderna e interactiva.

## Características

- Interfaz moderna y minimalista.
- Transiciones animadas entre Login y Registro.
- Formularios interactivos.
- Inputs con efecto floating label.
- Diseño centrado y limpio.
- Uso de JavaScript para alternar paneles.
- Validación básica mediante atributos HTML.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionamiento

El proyecto contiene dos formularios principales:

- Login
- Registro de usuario

Cuando el usuario presiona uno de los botones:
- `Sign Up`
- `Login`

el contenedor principal cambia de posición mediante JavaScript y CSS, ocultando un formulario y mostrando el otro con una animación deslizante.
